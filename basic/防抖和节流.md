都是为了限制函数的执行次数

#### 防抖

在事件触发n秒后再执行回调，如果n秒内再次触发了事件，则重新计时

使用场景：按钮提交、搜索框输入时请求

第一版如下

```js
function debounce(func, wait) {
    let timer = null;
    return ...args => {
        timer && window.clearTimeout(timer);
        timer = window.setTimeout(() => {
            func.apply(this, args);
        }, wait);
    };
}
```

第二版如下，添加立即执行功能

```js
function debounce(func, wait, imediate) {
    let timer = null;
    return ...args => {
        timer && window.clearTimeout(timer);
        if (imediate) {
            timer = window.setTimeout(() => {
                timer = null;
            }, wait);
            // 初始先执行一次
            !timer && func.apply(this, args);
        }
        else {
            timer = window.setTimeout(() => {
                func.apply(this, args);
            }, wait);
        }
    };
}
```

#### 节流

事件触发的n秒内只触发一次函数，如果n秒内触发了多次事件，则只有一次生效

使用场景：拖拽、窗口resize等

第一版如下

```js
function throttle(func, wait) {
    let timer = null;
    return ...args => {
        if (!timer) {
            timer = window.setTimeout(() => {
                func.apply(this, args);
                window.clearTimeout(timer);
                timer = null;
            }, wait);
        }
    };
}
```

第二版如下，添加配置项，可以选择事件触发的初始是否执行回调，事件停止触发时是否执行回调

- leading：false 表示禁用第一次执行
- trailing: false 表示禁用停止触发的回调

```js
function throttle(func, wait, option) {
    let timer = null;
    let pre = 0;
    return ...args => {
        let now = new Date().getTime();
        if (!pre && option.leading === false) {
            pre = now;
        }
        const remaining = wait - (now - pre);
        // 如果在n秒之外的时候触发了事件或设置了leading：true，执行回调
        if (remaining <= 0 && remaining > wait) {
            if (timer) {
                window.clearTimeout(timer);
                timer = null;
            }
            pre = now;
            func.apply(this, args);
        }
        // 保证n秒内如果事件停止时可以触发回调
        else if (!timer && option.trailing !== false) {
            timer = window.setTimeout(() => {
                pre = option.leading === false ? 0 : new Date().getTime();
                timer = null;
                func.apply(this, args);
            }, remaining);
        }
    };
}
```
